# ADHD Pupillometry Analysis

Reproducing the key figures and statistics from **Wainstein, Rojas-Líbano, Crossley, Carrasco, Aboitiz & Ossandón (2017), "Pupil Size Tracks Attentional Performance in Attention-Deficit/Hyperactivity Disorder," *Scientific Reports* 7:8228.**

The dataset is not included here (it is ~3 GB and belongs to the original authors). Download it from figshare:
https://figshare.com/ndownloader/files/14298953
